1.4 离群点检测¶

IQR = Q3 - Q1
下限 = Q1 - 1.5*IQR
上限 = Q3 + 1.5*IQR
若 下限 <= x <= 上限， 则x非离群点，否则为离群点

3.5 利用相关关系填充¶

In [52]:
indexes = (data.price < 200) & (data.points.notnull())
X = (data.points[indexes]/100).values
y = (data.price[indexes]/200).values

newdata = data.copy()
# 线性回归
from sklearn import linear_model
model = linear_model.LinearRegression()
model.fit(X.reshape(-1, 1), y.reshape(-1, 1))

X_test = (data.points[(data.points.notnull()) & (~indexes)]/100).values
y_pred = model.predict(X_test.reshape(-1, 1))

newdata.loc[(data.points.notnull()) & (~indexes), 'price'] = y_pred*200
